# SpaceColonyApp

## Overview

The SpaceColony is an Android crew management game where you recruit, train, and deploy crew members on missions against space threats.

## Features

* 5 crew types (Pilot, Engineer, Medic, Scientist, Soldier)
* Move crew between Quarters, Simulator, and Mission Control
* Training system (XP increases skill)
* Turn-based missions (2 crew vs threat)
* Scaling difficulty system
* Permanent crew death

## Bonus Features

* Morale system (affects combat performance)
* Randomized combat outcomes
* Special abilities per crew type
* RecyclerView crew display
* Save and load system


## OOP Design

* Inheritance: CrewMember → subclasses
* Encapsulation: private fields
* Polymorphism: different crew behavior
* Strategy pattern: ActionSelector

## How to Run

1. Open project in Android Studio
2. Build the project
3. Run on emulator or device
